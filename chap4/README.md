## 算数运算
> 对于除法，C++新标准规定商一律向0取整
> 对于模预算，如果m%n不为0，则他的符号和m相同。
> m%(-n)等于m%n、(-m)%n等于-(m%n)


```cpp
	int m = -20;
	int n = -6;
	cout << m%n << endl;	//-2

	n = 6;
	cout << m%n << endl;	//-2

	m = 21;
	n = -5;
	cout << m%n << endl;  //1

```

## 练习4.4
>  向表达式添加括号，说明求值过程即最终结果。等于91.

```cpp
	int a = 0;
	a = 12 / 3 * 4 + 5 * 15 + 24 % 4 / 2;
	cout << a << endl;

	//取模运算优先级低于乘法和除法
	a = (12 / 3) * 4  + (5 * 15) + (24 % (4 / 2));
	cout << a << endl;
```
